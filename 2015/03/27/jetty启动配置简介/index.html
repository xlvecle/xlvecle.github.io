jetty是什么？

Jetty 是一个用 Java 实现、开源、基于标准的，并且具有丰富功能的 Http 服务器和 Web 容器。

在配置文件中会指定将要使用的XML文件，jetty.xml是默认的配置文件，该配置文件的根元素是Configure，另外还会看到有如下的配置元素：

  • Set: 相当于调用setXxx 方法
  • Get: 相当于调用getXxx 方法
  • New: 创建某个类的实例
  • Arg: 为方法或构造器传入参数
  • Array: 设置一个数组
  • Item: 设置数组或集合的项
  • Call: 调用某个方法

jetty.xml主要完成ThreadPool、连接器Connector、Handler等主要组件的配置，然后其他如jetty-deploy.xml、jetty-webapps.xml等文件中配置了一些其他内容。实际上，这些都写在一个配置文件中也是没有问题的。
