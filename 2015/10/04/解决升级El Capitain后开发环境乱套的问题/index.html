10-04-2015

升级完 El Capitain 之后开发环境发生了一定程度的乱套( 忍住,不喷 ), 出现了一些目录的权限问题以及 Xcode developer tools 失效的问题. 症状包括但不限于 homebrew 不能用, git 不能用, 终端配置失效. 一个典型的报错如下:

原因:
