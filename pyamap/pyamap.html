pyamap.amap module¶

class pyamap.amap.Amap(show_html=True, width='70%', height=300)¶

基类：object

pyamap旨在提供一个可以在notebook中方便绘制简单地理信息的python库，功能简单，易于使用

实际案例

>>> amap = from pyamap.amap import Amap
>>> amap = Amap()
>>> amap.setZoomLevel(15)
>>> amap.width = "70%"
>>> amap.height = "300px"
>>> amap.addMarker(116.397428,39.90923)
>>> amap.show()
